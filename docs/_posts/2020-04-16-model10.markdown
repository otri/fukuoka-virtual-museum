福岡藩の初代藩主・黒田長政（1568～1623）の兜です。兜の両脇の金色の角は，水牛の角を形どっています。長政の兜として有名な「一の谷兜」とともに，重要文化財に指定されています。

兜の正面に金色の丸い飾りがついた大水牛脇立兜と比べると，両脇の角が細く，シャープな印象です。


*大福岡城展（10月8日～11月29日）で展示予定
